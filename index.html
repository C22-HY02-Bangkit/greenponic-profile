About Greenponic

Greenponic is the HY-02 team from Indosat Ooredoo Hutchison which consists of 9 members, each path consists of 3 members. Greenponic focuses on making monitoring applications for hydroponic plants for making it easier to monitor hydroponic plants remotely in real-time.

Hydroponics could be started by utilizing the soil and become a hobby in the spare time. Although it sounds easy to get started, we have to focused and pay our attention to many aspects for managing hydroponics. Those aspects are hydroponic plant variables include TDS, pH, EC, and the threshold of each plant. Of course this will be difficult if we do not used to it, so we need a system that could help in managing hydroponics. Therefore, we provide Greenponic as an application that could help us managing hydroponics remotely.

Features

There are several features contained in our team's application that will be used to monitor hydroponics.

USER AUTHENTICATION

There are 4 features; registration, login, forgot and reset password. Users could login by fill their email and password. Users will directed to the registration page if they do not have an account yet. Uses should fill their name, email, and password for signing up. There is the Forgot Password page if users forgot theirs. Users should fill their email and get a link that will directed to the Reset Password page.

DATA REALTIME

This page contains data and graph of the hydroponics. The data contains percentage of tds Censor and the amount of EC and pH Censor. The graph contains a graph that show tds, EC, and pH Censor.

CONTROLLER

Users could change the value of tds, EC, and pH Censor on their own in the controller. This feature can help users maintain their hydroponics remotely.

GREEN MEDIA

This feature same like Social Media, users could posting or share their hydroponics information, comment to another users post, and follow other users. This feature can help users maintain their yields.

E-COMMERCE

Admin and users could see all of the products, get the detail of each product, add and edit a product, give rating to a product, then review the product and update it on their own.

NOTIFICATION

Users could see the notification they received for each day. Users will know if there is an anomaly detected in their plants and if their plants need more vitamins. Users also get a notification if someone is following their account.
